\doxysubsection{Descripción detallada}
Clase que representa un cuadro de diálogo para agregar información de una tarjeta. 

La clase \mbox{\hyperlink{classAddCardDialog}{Add\+Card\+Dialog}} hereda de QDialog y proporciona un diálogo en la interfaz gráfica para ingresar detalles como nombre, apellido, DNI, email, teléfono, dirección, edad, información de licencia de vehículo y datos del vehículo (marca, modelo, color y tipo).

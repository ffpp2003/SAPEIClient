\doxysubsection{Descripción detallada}
Clase que maneja la comunicación serie con un dispositivo (por ejemplo, Arduino). 

La clase \mbox{\hyperlink{classSerialHandler}{Serial\+Handler}} permite seleccionar un puerto serie, conectarse a él, enviar mensajes, y recibir datos desde el dispositivo conectado. También proporciona una señal que emite cuando se recibe un ID.

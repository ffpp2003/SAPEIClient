\doxysubsection{Descripción detallada}
Clase principal que maneja la interfaz gráfica y la lógica de la aplicación. 

La clase \mbox{\hyperlink{classMainWindow}{Main\+Window}} es responsable de interactuar con el usuario, manejar la recepción de IDs a través del puerto serie, y gestionar la base de datos de tarjetas. Además, permite agregar nuevas tarjetas y seleccionar el puerto serie.
